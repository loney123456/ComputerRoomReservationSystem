# ComputerRoomReservationSystem
高校机房预约系统

编译运行环境：VS2019 IDE、Window10

（SRC文件夹中是源代码，Release文件夹中是编译好的可以直接运行的可执行文件）

主要涉及的技术点：map容器、vector容器、文件读写操作、C++中封装、继承、多态的特性
